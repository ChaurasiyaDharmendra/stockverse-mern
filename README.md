# StockVerse 🚀

StockVerse is a full-stack stock trading dashboard inspired by Zerodha, built using the MERN stack.

## 🔥 Features
- User Signup & Login
- Dashboard with holdings & positions
- Orders & portfolio tracking
- Interactive UI

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## ⚡ How to Run
```bash
npm install
npm start
